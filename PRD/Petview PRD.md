# Petview PRD

## **界面布局与功能**

Core idea of the function: 

<aside>
💡

- **顶部区域**
    - **左侧是状态栏（图标+数值）**：
        - 用户当前能量值（初始为0），狐狸的关系值（初始为30），经验值（初始LV 1: 0/10）
        - 每天0点如果前一天没有完成任何任务，关系值下降2
        - 关系值最低不低于15
    - **右侧是返回键：**进入more界面
- **中央互动区zstack**
    - **宠物狐狸**：
        - 狐狸根据关系值高低在花丛间切换状态。
        - 当前为静态图片+屏幕下方一半范围内随机移动。
        - 点击狐狸并上下滑动可以完成抚摸动作，关系值+1。同一天内限三次。
        - optional：静态图片切换为动图或者视频
    - **装扮商店**：
        - 点击礼物盒进入商店。通过获得的能量值，用户可以购买不同的装扮类型（decor，clothing，shoes，food）。
        - optional：部分装扮需要经验等级提升后解锁。
        - 装扮购买后按钮改为tickbox，可以把装扮应用到狐狸形象。同一个类型只允许tick一个。
        - 购买新装扮后关系值+10，经验值+10。
        - 食物无法购买，只能通过完成任务获得，图标下方显示数量。喂食可直接点击食物图标。每次喂食后关系值+1，经验值+1。
    - **任务火箭按钮**
        - 画面右侧悬浮一枚任务飞船图标，象征用户外出探险（用户现实生活）。点击打开“任务”页面。这里列出今日任务和tickbox。
        - 每项完成后经验值+1，并且根据任务系统规则获得能量值和物品。
        - 经验等级每次升级后数值重置为0，并且不累计前一级数值。1-5级解锁需要数值：10，25，50，75，100。后续每一级为100。
        - optional：当用户完成任务后，狐狸出现撒花效果
        - optional：经验升级除了解锁装扮以外，可解锁动作（撒娇），象征狐狸逐渐被“驯养”。
- **optional：**
    - **背景动态**：
        - 宠物狐狸的页面时间与用户实时时间**不同步**
        - 昼夜时间是3-15小时的随机时长。根据昼夜切换天空色调：白天是bg-main，晚上是bg-night。
        - 天气随机变化：晴天是main；多云是dawn；雨天加斜的下雨动态效果；雪天加下雪的动态效果，并且main渐变切换成snow。
        - 用户可以通过按钮手动切换天气。
    - **背景音乐**
</aside>
